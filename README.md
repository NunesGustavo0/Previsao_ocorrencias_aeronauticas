# Ocorrências Aeronáuticas

## Problema

Prever a seriedade de uma ocorrência de acidente utilizando a base de dados de ocorrências aeronáuticas da ANAC (Agência Nacional de Aviação Civil).

## Métrica de Sucesso

O projeto será considerado bem-sucedido ao atingir, no mínimo, **90% de F1-macro**.

## Cliente Fictício

Companhia de aviação interessada em analisar os principais fatores associados à ocorrência de acidentes, como:

- Conceitos relacionados às ocorrências;
- Modelos de aeronaves;
- Localidades;
- Condições operacionais;
- Outros fatores relevantes presentes na base de dados.
